# amit-t/skills category vocabulary

Use these exact category names when a repo authors reusable skills for the `amit-t/skills` catalog:

- `Product Management`
- `Project Management`
- `Engineering`
- `UX Design`
- `Agent Behavior`
- `AI Agent`
- `Leadership`

## Skill file convention

Each skill directory contains:

- `SKILL.md` — required instructions with frontmatter `name` and `description`.
- `README.md` — required install and usage guide.

## Catalog sync reminder

Only apply catalog sync rules in repos that publish a skills catalog. Ordinary consuming repos should keep consumer guidance only.
