# write-a-prd

> Create a PRD through interview, codebase exploration, and module design

**Category:** Product Management

## Install

Install using the [`skills`](https://github.com/vercel-labs/skills) CLI:

```bash
npx skills@latest add amit-t/skills --skill write-a-prd
```

Install all skills from this repository:

```bash
npx skills@latest add amit-t/skills
```
