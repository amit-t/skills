# request-refactor-plan

> Create a detailed refactor plan with tiny commits via interview, filed as GitHub issue

**Category:** Engineering

## Install

Install using the [`skills`](https://github.com/vercel-labs/skills) CLI:

```bash
npx skills@latest add amit-t/skills --skill request-refactor-plan
```

Install all skills from this repository:

```bash
npx skills@latest add amit-t/skills
```
